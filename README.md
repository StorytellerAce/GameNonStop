# 🎨 y2s1-web-app

An AI-powered, interactive storytelling and assistance platform built for a Year 2, Semester 1 web development project. This app combines modern UI design, real-time web scraping, and generative AI to provide users with smart guidance and engaging interactions.

---

## 🚀 Tech Stack

### Frontend

- **React** (v18) – Component-based UI
- **Vite** – Lightning-fast build tool
- **TailwindCSS** + **Bootstrap** – Utility-first and responsive design
- **React Router DOM** – Page routing
- **React Slick** – Carousel & sliders
- **SweetAlert2** – Better user prompts and confirmations

### Backend & Integration

- **Axios** – API and scraping requests
- **Google Generative AI API** – Text generation for AI Assistant
- **EmailJS** – Contact form submission without backend

### Tooling

- **ESLint & Prettier** – Linting and code formatting
- **PostCSS, Autoprefixer** – CSS preprocessing
- **GH Pages** – Deployed to GitHub Pages

---

## ✨ Notable Features

- 🔍 **Web Scraping Integration**  
  Scrapes external data dynamically and displays it to the user in real time (e.g. book data, summaries, etc.).

- 🤖 **AI Assistant (Bottom-Right Chatbot)**  
  A built-in chatbot that uses a language model (Google GenAI) to assist users, answer questions, and enhance interactivity.

- 🧠 **Clean, Beginner-Friendly Layout**  
  UI is designed for clarity and ease of use, especially for new users. Intuitive navigation with clear call-to-actions.

- 📨 **Contact Form**  
  Users can send inquiries directly via EmailJS without the need for server infrastructure.

- 🧼 **Well-Structured Component Design**  
  Organized code and interface layout using React components, promoting maintainability and scalability.
